Este repositório é um fork do projeto Sopra-Fitas, com várias modificações e melhorias feitas por mim.

O que foi alterado

Adicionado sistema de login e usuários

Criação de perfil do usuário, com edição de nome, usuário, email e senha

Importação de ROMs por pasta, direto do computador do usuário

Biblioteca de jogos individual por usuário

Jogos podem ser privados ou enviados para o público

Jogos públicos passam por aprovação do administrador

Painel admin para gerenciar jogos e usuários

Detecção automática do console pela extensão da ROM

Busca automática de nome, descrição e capa via Wikipedia (PT-BR)

Interface ajustada e melhor organizada

Observações

O projeto não distribui ROMs ou BIOS

As ROMs ficam salvas localmente no navegador do usuário

Projeto com foco educacional e experimental

Tecnologias

React

Vite

EmulatorJS

JavaScript

npm install
